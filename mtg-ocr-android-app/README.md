# MTG OCR Android App

This project is an Android application designed to capture images of Magic: The Gathering cards and perform Optical Character Recognition (OCR) using the Gemini Flash Lite API. The app recognizes card details such as name, language, collector number, set code, and year of print.

## Features

- **Camera Functionality**: Capture images of cards with an adjustable camera view.
- **OCR Processing**: Utilize the Gemini Flash Lite API to extract card details from images.
- **Card History**: Save and manage a history of found cards with options to delete entries.
- **Scryfall Integration**: View captured card images and search for additional information on Scryfall.
- **Export to CSV**: Save card data locally in CSV format for easy access and sharing.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd mtg-ocr-android-app
   ```

3. Open the project in Android Studio or your preferred IDE.

4. Sync the project with Gradle files to download dependencies.

5. Connect an Android device or start an emulator.

6. Run the application.

## Usage

- Launch the app and use the camera feature to capture images of Magic: The Gathering cards.
- The app will process the image and display recognized card details.
- Access the history feature to view previously captured cards and manage your collection.
- Use the export functionality to save your card data in CSV format.
